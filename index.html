Que es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y diseñar páginas web. Es el lenguaje base sobre el cual se construyen la mayoría de los sitios web en internet. HTML utiliza una estructura de etiquetas para definir la estructura y el contenido de una página web. Cada elemento en una página web, como texto, imágenes, enlaces, formularios, etc., se define y se coloca utilizando etiquetas HTML. Estas etiquetas proporcionan instrucciones al navegador web sobre cómo renderizar y mostrar el contenido de la página al usuario.

para que sirve los comandos basicos de HTML

  • Etiqueta "html": Define el principio y el final del documento HTML. Todo el contenido de la página web debe estar dentro de esta etiqueta.

  • Etiqueta "head": Contiene metadatos sobre el documento HTML, como el título de la página, enlaces a hojas de estilo CSS, scripts JavaScript, metadatos SEO, etc.

  • Etiqueta "title": Define el título de la página web, que se muestra en la barra de título del navegador.

  • Etiqueta "body": Contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, etc.

  • Etiqueta "h1", "h2", "h3", ..., "h6": Son etiquetas de encabezado que se utilizan para definir títulos y subtítulos en la página. "h1" es el título más importante, seguido de "h2" como subtítulo, y así sucesivamente.
  • Etiqueta "p": Define un párrafo de texto en la página.

  • Etiqueta "a" (ancla): Se utiliza para crear hipervínculos. El atributo href especifica la URL a la que el enlace debe dirigir. Por ejemplo:

  • Etiqueta "img": Se utiliza para incrustar imágenes en la página. El atributo src especifica la ruta de la imagen y el atributo alt proporciona un texto alternativo que se muestra si la imagen no se puede cargar. Por ejemplo:

  • Etiqueta"ul", "ol", "li": Se utilizan para crear listas sin ordenar (puntos(ul)) y ordenadas (números o letras(ol)).

  • Etiqueta"table","tr","td":Se utilizan para crear tablas y definir filas y celdas dentro de ellas.

  • Etiqueta"form","input","button","textarea": Se utilizan para crear formularios de entrada de datos.

  • Etiqueta"div","input":Se utilizan para agrupar y dividir secciones de contenido en la página.

Estos son los comandos basicos de HTML que ya te permiten hacer una pagina basica.
